# GRASSP SQL Sprint

This repo contains a PDF of GRASSP Acad 30 days SQL scenarios along with questions, sample data to create and insert,
and what concepts each scenario question cover.

**How to Use This Sprint**

Each day follows the same format: a real workplace scenario at SuperFastFood Global (SFF-G), the business
question behind it, sample data you can run yourself in PostgreSQL, the query, and a breakdown of exactly what
SQL concept it tests.

**SUGGESTED APPROACH:**
* Read the scenario and try to write the query yourself before looking at the answer.
* Run the CREATE TABLE and INSERT statements in PostgreSQL (or any SQL engine of your choice) to test your own
query.
* Compare your query against the provided solution and read the "What This Tests" breakdown.
* Pay attention to the recurring patterns — CTE + window function, conditional aggregation, LEFT JOIN for 
completeness — these show up again and again in interviews.

Go through each and every question and practice along these 30 Real Business SQL Questions based on fictional fast food chain dataset "SuperFastFood Global (SFF-G)". This covers beginner to adavnced topics which helps you to attend any SQL interview with confidence.
